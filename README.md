# WooHoo - TouchDesigner Project

This repository is set up to collaborate on TouchDesigner scripts and shaders.

## Project Structure
- `/scripts`: Python scripts for DATs.
- `/shaders`: GLSL shader code.
- `/assets`: External media and data files.

## How to use with TouchDesigner
1. **Clone** this repository to your local PC.
2. In TouchDesigner, use the **Text DAT** or **GLSL TOP** to point to external files in this folder.
3. Enable **"Sync to File"** in the DAT/TOP parameters to automatically reload changes when I push updates.
4. Let me know which script or shader you want me to work on, and I will push the code here!
